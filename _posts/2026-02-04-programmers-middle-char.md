## 문제 요약
문자열 `s`가 주어질 때 가운데 글자를 반환한다.

- 길이가 홀수면 가운데 1글자
- 길이가 짝수면 가운데 2글자

## 접근 과정
### 1) 첫 생각
문자열 길이를 `n`이라고 하면 가운데 인덱스는 `n // 2`이므로 이를 기준으로 처리한다.

### 2) 막힌 지점
짝수 길이에서 정확히 2글자를 가져오려면 어떤 범위를 슬라이싱해야 하는지 잠깐 헷갈렸다.

### 3) 해결 전략
`mid = n // 2`로 두고,
- 홀수면 `s[mid]`
- 짝수면 `s[mid-1:mid+1]`
로 정리했다.

## 풀이 코드

~~~python
def solution(s: str) -> str:
    n = len(s)
    mid = n // 2

    if n % 2 == 1:
        return s[mid]
    return s[mid - 1: mid + 1]
~~~

## 복잡도 분석
- 시간복잡도: O(1)
- 공간복잡도: O(1)

## 회고
- 짝수 케이스에서 인덱스 경계를 예제로 먼저 확인하는 습관이 필요하다.
- 슬라이싱은 끝 인덱스가 포함되지 않는다는 점을 항상 의식해야 한다.
- 다음엔 길이 3/4 같은 최소 예제를 먼저 적고 구현하겠다.
