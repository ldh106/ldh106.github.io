## 🧾 문제 요약
- 문자열 s의 가운데 글자를 반환  
- 길이가 홀수면 1글자  
- 길이가 짝수면 2글자

## 📌 예제 정리

입력 예시:
~~~
"abcde"
~~~

출력 예시:
~~~
"c"
~~~

입력 예시:
~~~
"qwer"
~~~

출력 예시:
~~~
"we"
~~~

## 🧠 접근 과정

### 1) 첫 생각
- 문자열 길이를 기준으로 가운데 인덱스를 구하면 해결 가능하다고 판단
- 파이썬 슬라이싱을 사용하면 조건 분기가 간단할 것이라 생각

### 2) 막힌 지점
- 짝수 길이일 때 정확히 어떤 범위를 잘라야 하는지 헷갈림
- mid 인덱스 기준으로 좌우 한 칸씩 가져와야 한다는 점 정리 필요

### 3) 해결 전략
- n // 2 를 기준으로  
- 홀수 → s[mid]  
- 짝수 → s[mid-1:mid+1] 로 처리

## 💻 풀이 코드 (Python)

~~~python
def solution(s: str) -> str:
    n = len(s)
    mid = n // 2

    if n % 2 == 1:
        return s[mid]
    return s[mid - 1: mid + 1]
~~~

## ⏱ 복잡도 분석
- 시간복잡도: O(1)
- 공간복잡도: O(1)
